Esta semana 14, ademas de realizar varios ejemplos para familiarizarnos con las funciones que nos ofrece python, también realizamos un programa en python que realiza una simulacion de un oscilador armonico simple (por ejemplo, un pendulo simple). Esto usando el metodo de euler. Se tomaban condiciones iniciales de velocidad, amplitud y frecuencia, para despues realizar una simulacion en n segundos dados. Las velocidades y posiciones a cada momento fueron guardadas en dos respectivos arreglos, para despues realizar 3 graficas:

1. Posición vs. tiempo
2. Velocidad vs. tiempo
3. Posición vs. velocidad
